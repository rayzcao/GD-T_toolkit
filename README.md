# GD-T_toolkit

1. Positional Tolerance Calculator: To calculate true position error and pass/fail status.<br>

2. Bonus Tolerance Calculator: To compute extra tolerance available at Maximum Material Condition (MMC) or Least Material Condition (LMC).<br>

3. Symbol Library: A quick reference guide for standard GD&T symbols.<br>

4. Visualizations: Interactive SVG diagrams to visualize the tolerance zones.
